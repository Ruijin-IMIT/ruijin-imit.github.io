分割任务评分方法

1. 评估指标

对双侧半球(左和右)的每个脑区，我们计算：

  • Dice系数(Dice): 衡量预测与真实标签的重叠度(权重: 0.7)
  • Hausdorff距离(Hausdorff): 衡量边界一致性(权重: 0.3)

2. 权重方案

类别 权重 描述
模态权重 QSM: 0.6
NM: 0.4		 每种成像模式的相对重要性
QSM脑区权重 尾状核: 1.0
壳核: 1.0
苍白球: 1.0
丘脑底核: 1.2
黑质: 1.5
红核: 1.1
齿状核: 1.0 		QSM中每个脑区的相对重要性
NM脑区权重 黑质: 2.0 NM中脑区的相对重要性
指标权重 Dice: 0.7
Hausdorff: 0.3		 每个评估指标的相对重要性

3. 评分公式

团队的综合得分计算如下：

综合得分 = 0.6 × QSM得分 + 0.4 × NM得分

QSM得分 = 1.0×尾状核 + 1.0×壳核 + 1.0×苍白球 + 1.2×丘脑底核 + 1.5×黑质 + 1.1×红核 + 1.0×齿状核
NM得分 = 2.0×黑质

其中每个脑区得分(如尾状核)计算如下：
脑区得分 = 0.7×Dice + 0.3×(1 - 标准化Hausdorff)
标准化Hausdorff = Hausdorff / 最大可能Hausdorff
(最大可能Hausdorff = 10)

4. 排名标准

团队按综合得分降序排名。最高可能得分为9.8(所有脑区完美分割)。
