Evaluation Criteria

Nuclei Segmentation Task

  • Dice coefficient used to evaluate segmentation accuracy of 7 Parkinson's disease-related nuclei
  • Final score is the average Dice score across all nuclei
  • Validation set (100 cases) for development phase feedback
  • Test set (200 cases) for final ranking determination
  • Imaging modalities: Neuromelanin-sensitive MRI (NM-MRI) and QSM

PD Classification Task

  • Classification accuracy used to evaluate PD diagnosis performance
  • Validation set (100 cases) for development phase feedback
  • Test set (200 cases) for final ranking determination
  • Binary classification: Parkinson's disease vs. healthy controls
